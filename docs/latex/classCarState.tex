\doxysubsection{Detailed Description}
Class that contains the state of the car. 

The \mbox{\hyperlink{classCarState}{Car\+State}} class represents the state of a car in terms of its position (x, y), orientation (theta), and provides methods to calculate new states, check validity, and compute rewards based on the car\textquotesingle{}s configuration.
