Detailed Description

The MainWindow class represents the main interface for the application.

This class inherits from QMainWindow and is responsible for handling user interactions, managing the environment, and visualizing the state of the Q-learning model and car simulation. It contains methods to control the simulation and manage the Q-learning process.
